# QuickList Web Application
Aplikasi yang digunakan untuk mencatat berbagai barang yang dibutuhkan saat sedang berbelanja.

![image](https://github.com/user-attachments/assets/a7eb4e7c-1fb1-4d97-b017-f26618d879b6)

## QuickList Featured
<ul>
  <li>Tambah & hapus daftar barang.</li>
  <li>Centang datar barang untuk mengetahui barang sudah dibeli.</li>
  <li>Sorting daftar barang.</li>
  <li>Hitung presentasi barang belanjaan yang sudah dibeli.</li>
</ul>

# React + Vite

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

Currently, two official plugins are available:

- [@vitejs/plugin-react](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react/README.md) uses [Babel](https://babeljs.io/) for Fast Refresh
- [@vitejs/plugin-react-swc](https://github.com/vitejs/vite-plugin-react-swc) uses [SWC](https://swc.rs/) for Fast Refresh
